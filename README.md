# Django-Academic-Example

## Description

This project is a simulated platform for a school, built with Django. The platform allows managing information about students, enrollments, and courses, storing the data in a MySQL database.

## Key Features

- Student registration with information such as name, enrollment, etc.
- Management of courses and student enrollments.
- Persistent storage of data in a MySQL database.
- Administration interface for easy data management.
- Contact form that sends messages to a specific email.

### Install dependencies:

pip install -r requirements.txt
Configure the database in settings.py.

python manage.py migrate
### Start the development server:

python manage.py runserver

### Usage
Access the platform from your browser: http://localhost:8000/
Use the contact form to send messages to the specific email.
